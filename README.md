# Legal CV Skill

This project is focused on developing a comprehensive legal CV.

## Technologies
- JavaScript
- Node.js
- Express
- MongoDB

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Adriano1976/legal-cv-skill.git
   cd legal-cv-skill
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up the environment variables as needed.

## Testing Information
To run tests, use the following command:
```bash
npm test
```
